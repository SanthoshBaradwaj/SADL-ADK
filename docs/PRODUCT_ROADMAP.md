# SADL Product Roadmap

## MVP

- Repo template.
- CLI init/adopt/status/validate/checkpoint/manifest/dream/commit.
- Schemas for config, intake, roadmap states, and session logs.
- Secret-safe defaults.
- Agent prompt templates.
- Multi-agent policy docs.
- Sample project use cases.

## Next

- Interactive intake wizard that writes PRD and architecture files.
- JSON schema validation without external dependencies or with optional Ajv integration.
- Configured validation command runner.
- Git worktree helper for multi-agent branches.
- CI action for `sadl validate`.
- Adapter generators for Codex, Claude Code, Cursor, Kiro, Gemini, GitHub Copilot, and Devin-style agents.
- Dashboard for active task, approvals, token/cost metadata, and session outcomes.

## Later

- MCP server.
- IDE extension.
- Policy-as-code integration.
- Enterprise audit exports.
- Team analytics for cost, rework, and validation pass rate.
