# Kiro Adapter

## Setup

Kiro-style spec, steering, and task files overlap with SADL concepts. Keep SADL docs as the cross-tool source of truth.

Suggested mapping:

```text
SADL PRD         -> Kiro spec requirements
SADL ARCH_SPEC   -> Kiro steering
SADL ROADMAP     -> Kiro tasks
SADL STATE       -> session handoff
```

Do not duplicate conflicting instructions. If Kiro steering and `AGENTS.md` disagree, pause and ask a human to reconcile them.
