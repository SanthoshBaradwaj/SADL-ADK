# SADL Adapters

Adapters are thin compatibility layers. They should not become the source of truth. The source of truth remains the repository protocol:

```text
AGENTS.md
.sadl/config.json
docs/01_PRD.md
docs/02_ROADMAP.md
docs/03_STATE.md
docs/04_ARCH_SPEC.md
docs/05_SESSION_LOG.md
```

Use these files as setup notes for specific assistants or IDEs.
